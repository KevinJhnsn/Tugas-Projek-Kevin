# **App Name**: Bijak Rupiah

## Core Features:

- Secure Login: Authenticate users securely with Google Identity Services.
- Goal Setting: Create and track personalized savings goals with due dates.
- Expense Tracking: Record daily income and expenses for detailed financial monitoring.
- Savings Visualization: Display savings progress through charts and graphs, helping users stay motivated.
- AI Financial Insight Tool: AI-driven insights into spending habits, providing personalized financial advice and suggesting ways to save more effectively. It predicts when savings goals will be met, and may use tool that detects applicable savings tips. Defaults user's name to 'Kevin'.
- Responsive Design: Ensure seamless functionality and optimal display across laptops, tablets, and smartphones.
- Persistent Data Storage: Reliably store user financial data in a robust and scalable database system. Database can be utilitized by end users and is optimized.

## Style Guidelines:

- Primary color: Fresh green (#77DD77), reflecting growth and financial well-being.
- Background color: Light, desaturated green (#E2F4E2) to provide a clean and calming backdrop.
- Accent color: Light blue (#ADD8E6) complements the green, signifying trust and clarity.
- Body and headline font: 'PT Sans', a humanist sans-serif for a blend of modern and approachable design.
- Use clear, modern icons to represent different categories of income and expenses.
- Prioritize a clean, intuitive layout for easy navigation and data input.
- Subtle transitions and animations to provide feedback and enhance user experience.
